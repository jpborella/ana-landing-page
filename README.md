# Landing Page - Ana Paula Farias



Uma landing page simples e moderna criada para apresentar produtos através de um QR Code — ideal para vendas presenciais rápidas. Projetada com React + TypeScript + Tailwind CSS e Vite.


## Principais características

- Layout responsivo usando Tailwind CSS
- Catálogo de produtos com cards reutilizáveis
- Modal de detalhes do produto

## Tecnologias

- React 18
- TypeScript
- Tailwind CSS
- Vite
- FontAwesome (ícones)

## Requisitos

- Node.js v16+ e npm (ou yarn/pnpm)

## Instalação (rápido)

Abra o terminal na raiz do projeto e execute:

```powershell
cd
npm install
npm run dev
```
